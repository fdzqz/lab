这是学院一个科研团队（数据与决策工程重点实验室）的官网，主要功能是展示研究团队的基本情况、成员介绍等等。

## 使用技术

### 前端
使用了React来搭建View层，但是没有做成单页应用，每个页面单独构建，并使用了React-router来管理路由。

### 后台
使用Express作为后台框架，MongoDB作为数据库。
